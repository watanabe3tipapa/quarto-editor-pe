# Quarto Editor PE - 使い方ガイド

インタラクティブなQuartoドキュメントエディタの基本的な使い方説明します。

## ファイル操作

### ドキュメントを読み込む

1. ヘッダーの「📂 UPLOAD」ボタンをクリック
2. `.qmd`、`.md`、`.txt` ファイルを選択
3. エディタに内容が表示されます

### ドキュメントを保存する

1. ヘッダーの「💾 SAVE」ボタンをクリック
2. `quarto-document.qmd` としてダウンロードされます

### ドキュメントをクリアする

1. ヘッダーの「🗑️ CLEAR」ボタンをクリック
2. 確認ダイアログが表示されるので「OK」をクリック
3. エディタの内容がすべて削除されます

---

## 編集とプレビュー

### Editモード

- 左側の「Edit」ボタンをクリック
- Markdownドキュメントを編集できます
- シンタックスハイライト、コード補完、自动インデントが有効です

### Previewモード

- 左側の「Preview」ボタンをクリック
- 整形されたドキュメントのプレビューが表示されます
- コードブロックがハイライト表示されます

---

## コードブロックの実行

### 基本的な流れ

1. コードブロックを見つけます（灰色の背景で表示）
2. ブロック横の「▶ 実行」ボタンをクリック
3. ステータス表示が変化します

### ステータス表示

| 表示 | 意味 |
|------|------|
| ○待機中 | 実行前 |
| ●処理中 | 実行中（ボタン無効） |
| ✓成功 | 実行完了 |
| ✗エラー | 実行エラー |

### 結果の確認

**成功時:**
- 緑色の枠で結果が表示されます
- 「実行完了」などのメッセージ

**エラー時:**
- 赤色の枠でエラー情報が表示されます
- エラーの種類に応じたヒントが表示されます
- 「▼ 詳細を表示」をクリックすると技術的な詳細を確認できます

### リセット

「Reset」ボタンをクリックすると、実行結果をクリアし、待機中の状態に戻ります。

## AIアシスタント

AIを使ってコードの解説や補完ができます。

### を開く

画面左下の「🤖 AI」ボタンをクリックします。

### コードを選択する

1. エディタのコードブロックをクリックします
2. 選択したコードがAIアシスタントに渡されます

### AI機能の使い方

| 機能 | 説明 |
|------|------|
| 📖 Explain | コードの意味を日本語で説明します |
| ✨ Complete | コードの続きを提案します |
| 🔄 Refactor | コードをリファクタリングします |

### チャットで質問する

テキスト入力欄に質問を入力して送信できます。
AIがあなたの質問にお答えします。

### 接続状態

AIアシスタント右上のインジケーターで確認できます:

| 色 | 状態 |
|----|------|
| 緑 | Ollamaに接続済み |
| 赤 | 接続できません |
| 黄 | 確認中 |

## Ollamaについて

AIアシスタントはOllamaというローカルAIサーバーを使用します。

### 事前準備（ローカルで使用する場合）

```bash
# Ollamaをインストール（macOS）
brew install ollama

# AIモデルをダウンロード
ollama pull llama3.2

# Ollamaを起動
ollama serve
```

### Vercel版について

このアプリをVercelにデプロイしている場合、AIアシスタント機能は利用できません。
AI機能を使う場合は、ローカル環境でお楽しみください。

## トラブルシューティング

### ボタンが反応しない

- ページを更新してください
- ブラウザのコンソールにエラーがないか確認してください

### AIアシスタントに接続できない

1. Ollamaが起動しているか確認: `ollama serve`
2. モデルがダウンロード済みか確認: `ollama list`
3. 必要に応じてモデルをダウンロード: `ollama pull llama3.2`

### ファイルがアップロードできない

- 対応形式を確認してください: `.qmd`, `.md`, `.txt`
